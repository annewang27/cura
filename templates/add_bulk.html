{% block title %} Add new data through a json file{% endblock %}

This is a shortcut to add multiple collections, artists, or art at once. Make sure every piece of art is added to a collection that either already exists or is being added in the same JSON.

The JSON should contain a dictionary with three keys: collections, artists, art. To add items to a category, create a list containing the correct objects. For example, to add three paintings, create a list with 3 art objects as the value corresponding to the art key. If nothing is being added to a category, the dictionary key should still be there, but the corresponding value should be an empty list.

A collection should contain: title, and summary.

An artist should contain: full_name, dob, and summary.

An artwork should contain: title, artist, created, collection_name, filepath, and summary.
